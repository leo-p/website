# Website

Static personal website [leo-p.github.io/](https://leo-p.github.io/)based on [hugo](https://gohugo.io) and the [even theme](https://gohugo.io).

## Install
```console
$ brew install hugo
```

## Preview
```console
$ hugo server -D
```

## Build
```console
$ rm -rf *
$ hugo
```

## Deploy
```console
$ cd public
```
